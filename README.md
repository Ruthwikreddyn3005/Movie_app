# Movie App

A full-stack movie discovery and management web application built with React, Vite, Node.js, and MySQL.

## Live Demo

**[https://movie-app-ecru-six.vercel.app/](https://movie-app-ecru-six.vercel.app/)**

## Features

- User authentication (Login / Register)
- Browse and search popular movies via TMDB API
- Add / remove movies from personal favorites
- View and edit user profile (username, email, phone, bio, password)
- Persistent login across page refreshes

## Tech Stack

**Frontend:** React 19, React Router, Vite — deployed on Vercel
**Backend:** Node.js, Express — deployed on Railway
**Database:** MySQL — hosted on Railway
**APIs:** The Movie Database (TMDB)
